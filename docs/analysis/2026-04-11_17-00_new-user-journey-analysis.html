So what
  1. 1 FIX 'PERMISSION NOT GRANTED' ERROR (critical, this week) — 8 external users in 2 days hit this single error, equaling 62% of successful signup volume. The Google OAuth consent screen isn't pre-checking the Google Ads checkbox. Add a visual callout and a retry button on the error page. This alone could increase activation by 50%+.
  2. 2 CONVERT EXPLORERS WITH CONTEXTUAL SUGGESTIONS (biggest opportunity, ~2 weeks) — 4 of 13 new users (31%) queried data extensively but made zero changes. After 3+ read-only queries, surface: 'Based on your search terms, you could add 5 negative keywords to save ~$X/day. Want me to do it?' Push them past the activation threshold.
  3. 3 RESCUE STALLED USERS WITH TRIGGERED EMAIL (quick win, ~3 days) — 3 users connected and copied credentials but never ran an AI tool. User 10 browsed 12 pages and generated OAuth creds — they clearly want to use the product. Send a 'finish setup' email 24h after last activity with the auto-install command and a 2-min video.
  4. 4 FIX 'MISSING_CODE' AUTH FAILURE (3 users in 2d) — Likely an OAuth callback race condition. 3 external users hit it. Investigate the server-side callback handler and add automatic retry.
  5. 5 SEND WELCOME EMAIL WITH AUDIT RESULTS (catch bouncers, ~2 days) — 1 user signed up and vanished. Send within 1 hour: 'Your Google Ads account audit found 3 issues.' Give immediate value without requiring setup.
